Редактор переводов

В рамках данного проекта мной было разработано и реализовано веб-приложение "Редактор переводов" для моего личного использования (мой пет-проект).

Использованные технологии: PHP 8.1.17, Symfony 6.3.4, Twig 3.7.1, MariaDB 10.4.27

Клиентская часть: нативный javascript + jQuery, плагины jQuery, bootstrap 5

Приложение включает:

  1. удобный интерфейс для создания и редактирования переводов текстов на разные языки;
  2. каталог текстов по категориям и удобный интерфейс управления изменение;
  3. создание словарей по словам из текстов, над которыми осуществлялась работа. Поиск с использованием созданных словарей и добавленных в них слов.

Был реализован следующий функционал:

  1. Категории
    • вывод списка категорий с пагинацией и функционалом поиска
    • создание категории
    • редактирование категории
    • удаление категории
  2. Тексты
    • вывод списка текстов, относящихся к определенной категории, с пагинацией и функционалом поиска
    • создание текста
    • редактирование текста
    • удаление текста
    • вывод текста на отдельной странице и всех слов с вариантами перевода, встречающихся в этом тексте
  3. Словарь
    • вывод списка всевозможных словарей (например, английский - русский)
    • вывод списка слов для выбранного словаря со всевозможными значениями перевода с пагинацией и функционалом поиска
    • удаление слов со всеми значениями перевода
  4. Текст
    • вывод выбранного текста и всех слов с вариантами перевода, относящихся к этому тексту
    • удаление варианта перевода
  5. Главная страница
    • вывод добавленных на главную страницу текстов с переводом в виде панели с вкладками
    • изменение взаимного расположения вкладок с сохранением их порядка и активной вкладки
    • для каждой вкладки, соответствующей одному тексту доступно:
      • добавление слова в словарь
      • поиск по словарю*
      • редактирование текста и перевода с сохранением
      • вывод списка слов с переводом, добавленных в словарь из определенного текста
      • закрытие вкладки и соответственно удаление текста с главной страницы
      • на клиенте: при изменении текстов подсчет количества символов, очистка текста

* Поиск по словарю. Работая над переводом текста, пользователь имеет возможность добавить любое слово исходного текста в словарь, дописав его значение. Впоследствии это слово доступно: в общем словаре языка (здесь собираются все значения слова из разных текстов, по ссылке можно открыть текст с каждый соотвествующим значением), в словаре к каждому тексту. Доступно удаление как одного из значений, так и слова целиком. Также пользователь может осуществить, работая над любым текстом, поиск слов в этом тексте, которые ранее уже были добавлены в словарь. Приложение найдет все такие слова и добавит к ним список всех значений со ссылками на страницы текстов, в которых слово встречалось в соотвествующем значении.
